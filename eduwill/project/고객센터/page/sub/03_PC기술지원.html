내 PC
사양 확인

CPU/RAM 확인
- 바탕화면의 [내컴퓨터]라는 아이콘 위에서 마우스의 오른쪽 버튼을 클릭하고 속성을 선택합니다.
- 컴퓨터에 대한 기본 정보 보기 중 시스템에 있는 내용이 자신의 CPU와 RAM의 사양입니다.
- 자신의 컴퓨터 사양을 정확히 기억해 두세요. 문제 해결이 안될 경우, 자신의 컴퓨터 사양을 정확히 말씀해 주셔야 합니다.
비디오 메모리 확인
- 제어판에서 디스플레이 > 화면해상도 > 디스플레이 모양 변경에서 하단의 고급 설정을 클릭합니다.
- '어댑터' 탭에서 '칩 종류 /메모리 크기' 등이 자신의 비디오 메모리 환경입니다.
- 아래 표시된 내용을 반드시 기억해 두시고 문제가 발생하면 사양을 정확히 말씀해 주시기 바랍니다.
