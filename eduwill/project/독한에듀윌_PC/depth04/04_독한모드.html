나의 연속 수강 현황

이진욱님 독한모드 연속 수강에 도전하시겠습니까?
신청한 시간을 기준으로, 신청 이후 수강한 독한모드 수강 내역이 결과에 반영됩니다.

START 독한모드 보상안내

이진욱님은 2016-08-31부터 1단계(11일째) 진행중입니다.
지금 페이스 그대로, 매일 독하게 수강해주세요.

독한모드 보상안내 나의강의실로이동

이진욱님은 2016-08-31부터 1단계(11일째) 진행중(실패)입니다.
다시 도전하려면 RE-START 버튼을 클릭하세요

RE-START 독한모드 보상안내 나의강의실로이동
독한모드 이용안내
  1. 1. 독한모드 이벤트를 신청한다.
  2. 2. (신청일 포함) 매일 독한모드로 1강 이상 수강한다.
  3. 3. 각 단계 미션 완료 시, 활성화되는 보상 신청 버튼을 클릭하여
    　보상을 신청한다.
    　(현재 단계 완료 시, 보상 신청후 다음 단계 진행가능)
  4. 4. 다음 단계 보상을 획득하기 위해, 연속 수강을 진행한다.
독한모드 보상안내
  1. 1. 보상 혜택: 랜덤(비공개) 제공
  2. 2. 혜택 지급 방법
    　- 기프티콘 지급
    　- 회원정보에 기재된 휴대폰 번호로 발송
  3. 3. 혜택 지급일: 매월 말 일괄 발송
  4. 4. 각 단계별 혜택은 연속 1회씩 최대 총 4회 지급
  5. 5. RE-START(초기화) 후, 재시작하여 획득하는 보상 개수
    　제한 없음
  • ※ 1일 1회만 인정됩니다. (1일 기준, 중복 참여 불가능)
  • ※ 총 4단계까지 진행됩니다. (연속 4번 보상획득 기회 제공))
  • ※ 1번이라도 실패할 경우, 해당 단계 보상을 획득할 수 없으며 다음 단계 진행 영역이 활성화 되지 않습니다.
    　(RE-START로 재시작해야 함)
  • ※ RE-START(초기화) 시, 1단계 부터 다시 시작됩니다.
  • ※ RE-START(초기화)는 제한이 없으며, 재시작하여 획득하는 보상의 개수도 제한이 없습니다.
